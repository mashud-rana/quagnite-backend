Welcome to the Prometheus and Grafana sandbox environment.

The server called ubuntu-host has prometheus, a pushgateway and grafana running on it and monitors itself.

You can access the Prometheus and Grafana UI by clicking on the links above the terminal. A default dashboard called Process Metrics has been pre-configured for you. To access grafana, use the below credentials.

    User: admin
    Password: admin
Click the Red Cross button on the top right corner to end this sandbox environment

Feel free to explore and practice different commands. Happy coding!
