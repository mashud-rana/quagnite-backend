Welcome to the Ansible playground! This environment allows you to practice and explore Ansible. Sample playbooks and two servers node are set up for testing and playing around with.

Click the Red Cross button on the top right corner to end this sandbox environment

Basic Commands:

# Check the ansible version
ansible --version

# Running a Playbook
ansible-playbook httpd.yml


# Dry run (check mode)
ansible-playbook --check httpd.yml

Feel free to explore and practice different commands. Happy coding!
