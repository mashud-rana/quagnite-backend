Welcome to the Maven Sandbox Environment.
Maven is a build automation tool used primarily for Java projects but can also be used to build and manage projects written in other languages too..

To check if you have Maven installed run:
mvn --version.

Go to Project DIrectory :
cd maven-tutorial

Build the Project run:
mvn package

Click the Red Cross button on the top right corner to end this sandbox environment

Feel free to explore and practice different commands. Happy coding!
