Terraform Playground

Welcome to the Terraform playground! This environment allows you to practice and explore terraform in local mode. A sample terraform project is set up for testing and playing around with.


Click the Red Cross button on the top right corner to end this sandbox environment

Basic Commands:

# Check the terraform version
terraform version

# Initialize Terraform
terraform init
# Review changes
terraform plan
# Apply Changes
terraform apply

Feel free to explore and practice different commands. Happy coding!
